# Overview

This is my personal portfolio website showcasing my journey as a Full Stack Web Developer Trainee. The website features a modern, responsive design with a dark/light mode toggle based on system preferences. It highlights my skills, projects, and provides a way for potential employers or collaborators to get in touch.

# Features

- **Responsive Design** - Optimized for all screen sizes (mobile, tablet, desktop)
- **Dark/Light Mode** - Automatically adapts to system color scheme preferences
- **Interactive Project Cards** - Hover effects and clickable project links
- **Accessibility Features** - Skip links, ARIA labels, keyboard navigation support
- **Smooth Scrolling** - Enhanced navigation experience
- **Contact Form** - Fully functional contact form with validation
- **Social Media Integration** - Links to Instagram, GitHub, and LinkedIn
- **Print-Friendly** - Optimized print styles for physical copies

## Technologies Used
